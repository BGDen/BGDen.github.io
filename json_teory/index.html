XML разработан консорциумом World Wide Web в комитете, возглавляемом Джоном Босаком (Jon Bosak) в 1998 году. Он является упрощенной версией SGML.

В настоящее время XML широко используется для передачи данных между программами, он структурирует данные, организовывая их в легкую для разбора форму.

Если сравнивать их как структуру для передачи данных, то JSON лучше чем XML. Работать с объектной структурой намного легче, чем с DOM.

XML – это дополнение HTML.

  • Оба являются языками разметки.
  • Описание разметки выполняется при помощи тегов.
  • Теги могут иметь атрибуты.
  • Документы имеют иерархическую структуру (корневой элемент, вложенные элементы).
